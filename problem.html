Your code should output the text of T2 with highlighted elements corresponding to the matches for the regular expression of T1.

The solution must be written in HTML and JavaScript.

1. Mach the first occurrence only.

2. Match the last occurrence only.

3. Do not care about small and capital letters (i.e. do it case insensitive).

4. Match only at word boundaries.

5. Do not care about newlines (i.e. considering all the text glued rejecting any newline character).

6. Match all occurrences with overlapping
